CASE STUDY

SITUATION & TASK

Mengembangkan aplikasi asisten matematika cerdas yang mampu menyelesaikan soal dari input teks, suara, dan gambar (Vision).

Tantangan Utama: Mengatasi limitasi kuota API gratisan dan risiko halusinasi AI yang sering terjadi pada model matematika.

ACTION (SOLUTION)

Mengimplementasikan Triple-Fallback System (beralih otomatis antara Gemini 2.0 Lite, Flash, dan 1.5) untuk menjamin 99.9% uptime. Membangun sistem RAG (Retrieval-Augmented Generation) menggunakan SQLite untuk memori jangka panjang demi konsistensi jawaban.

RESULT (FEATURES)

  • Auto-generate PDF solution steps.
  • Audio explanation (Text-to-Speech).
  • Dynamic Graph Plotting using Matplotlib.
  • High-availability & Low-latency responses.
